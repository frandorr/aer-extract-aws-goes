# Technology Stack

- **Language**: Python 3.13+
- **Architecture**: Polylith (Bricks-based)
- **Package Manager**: [uv](https://github.com/astral-sh/uv)
- **Build System**: [Hatch](https://hatch.pypa.io/) with `hatch-polylith-bricks`
- **Linting & Formatting**: [Ruff](https://astral.sh/ruff)
- **Type Checking**: [Basedpyright](https://github.com/detachhead/basedpyright)
- **Frameworks**:
  - `aer-core`: Core plugin framework
  - `geopandas` / `pandera`: Geospatial data handling and schema validation
- **Testing**: `pytest`
