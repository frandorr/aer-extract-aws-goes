## Objective
Refactor the LUT storage in `lut.py` to use HuggingFace buckets directly with `fsspec` and Zarr, removing the old GitHub release downloading behavior.

## Tasks
1. Remove `download_lut` and `get_default_lut_release_url` from `lut.py`.
2. Add `get_default_bucket_uri()` pointing to `hf://bucket/frandorr/aer-data`.
3. Update `load_utm_zone_lut` to take a `bucket_uri` (or local path fallback) and use `fsspec.filesystem()` and `get_mapper` for `zarr.open`.
4. Update `core.py` to get the bucket URI from `extract_params` (or default) and pass it to `load_utm_zone_lut`.
5. Run tests to ensure everything still works.
